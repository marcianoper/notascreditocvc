# VDC360° · Notas a Crédito (Slots A/B + Reporte Diario)

Captura de comandas por cliente con productos y kilos, WhatsApp integrado,
edición/borrado en línea, reporte diario A4/CSV y protección de salida.

## Uso
Abre `index.html` en el navegador o visita GitHub Pages del repo.

## Config
Edita en `index.html`:
- SUPABASE_URL
- SUPABASE_ANON_KEY
- STORE_NAME
